DJI Telloドローンを制御するGoプログラムのコレクション

## 概要

このプロジェクトは、[Gobot](https://gobot.io/)フレームワークを使用してDJI Telloドローンを制御する複数のサンプルプログラムを含んでいます。

## 必要要件

- Go 1.25.4以上
- DJI Telloドローン
- WiFi接続（TelloドローンのWiFiネットワークに接続）

## インストール

```bash
go mod download
```

## プロジェクト構成

```
maeda_search/
├── chatGPT/
│   ├── main.go           # 右・左に移動する基本プログラム
│   └── usecase_1/
│       └── main.go       # 左・右移動 + 360度回転プログラム
├── claude_code/
│   ├── main.go           # 右・左に移動する基本プログラム
│   └── lib/
│       └── claude_code.go
├── go.mod
└── go.sum
```

## プログラム説明

### chatGPT/main.go

ドローンが以下の動作を実行します：
- 離陸
- 右に約20cm移動
- 左に約20cm移動（元の位置に戻る）
- 着陸

### chatGPT/usecase_1/main.go

ドローンが以下の動作を実行します：
- 離陸
- 左に約40cm移動
- 右に約40cm移動（元の位置に戻る）
- 時計回りに360度回転
- 着陸

### claude_code/main.go

ドローンが以下の動作を実行します：
- 離陸
- 右に約20cm移動
- 左に約20cm移動（元の位置に戻る）
- 着陸

## 使用方法

1. DJI TelloドローンのWiFiネットワークに接続します（通常は"TELLO-XXXXXX"という名前）

2. いずれかのプログラムを実行します：

```bash
# 基本的な右・左移動プログラム（ChatGPT版）
cd chatGPT
go run main.go

# 左・右移動 + 360度回転プログラム
cd chatGPT/usecase_1
go run main.go

# 基本的な右・左移動プログラム（Claude Code版）
cd claude_code
go run main.go
```

## 注意事項

- 実行前にドローンの周囲に十分なスペースがあることを確認してください
- 屋内での使用を推奨します
- バッテリー残量を確認してから実行してください
- 初めて実行する場合は、低い位置で試すことをお勧めします

## 依存ライブラリ

- [gobot.io/x/gobot](https://gobot.io/) v1.16.0 - ロボティクスフレームワーク
  - DJI Telloドライバーを含む

## ライセンス

このプロジェクトはオープンソースです。

## トラブルシューティング

### ドローンに接続できない場合

- TelloのWiFiネットワークに正しく接続されているか確認してください
- ファイアウォールがUDP 8890ポートをブロックしていないか確認してください
- ドローンの電源が入っているか確認してください

### プログラムがエラーで終了する場合

- Go 1.25.4以上がインストールされているか確認してください
- `go mod download`を実行して依存関係が正しくインストールされているか確認してください
